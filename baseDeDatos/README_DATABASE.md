# Scripts de Base de Datos - Proyecto Final BD2 CRMVentas

Este directorio contiene los scripts implementados para el proyecto final de Base de Datos II.

## Contenido

1. `01_funciones.sql` - Función escalar para calcular importe ponderado.
2. `02_procedimientos_almacenados.sql` - Procedimientos con transacciones explícitas.
3. `03_triggers_auditoria.sql` - Triggers de actualización y auditoría.
4. `04_indices_optimizacion.sql` - Índices no agrupados para optimización.
5. `05_vistas_reportes.sql` - Vistas de reportes operativos.
6. `06_datawarehouse_etl.sql` - Data Warehouse, ETL y vista analítica.
7. `07_mirroring_alta_disponibilidad.sql` - Guía de mirroring SQL1-SQL2.
8. `08_pruebas_validacion.sql` - Pruebas de validación.

## Orden sugerido

1. Ejecutar `creacionBDyTablas.sql`.
2. Ejecutar `01_funciones.sql`.
3. Ejecutar `02_procedimientos_almacenados.sql`.
4. Ejecutar `03_triggers_auditoria.sql`.
5. Ejecutar `04_indices_optimizacion.sql`.
6. Ejecutar `05_vistas_reportes.sql`.
7. Ejecutar `06_datawarehouse_etl.sql`.
8. Configurar alta disponibilidad con `07_mirroring_alta_disponibilidad.sql`.
9. Validar con `08_pruebas_validacion.sql`.
